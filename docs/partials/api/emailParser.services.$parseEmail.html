$parseEmail
service in module emailParser

Description

RFC822 mail parser service.

Usage

$parseEmail(mail, strict, parse);

Parameters

ParamTypeDetails
mailString
  • Email source in plain text
strictboolean
  • true for strict RFC822 compliance (don't treat \n without \r as line breaks)
parseboolean
  • false to disable actually parsing headers (just separate the header block and the body)

Returns

object
  • Parsed result containing headers and body properties.
